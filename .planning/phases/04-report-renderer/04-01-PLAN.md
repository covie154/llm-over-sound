```python
"""Report renderer: assembles formatted plain text reports from templates and findings.

Implements a base ReportRenderer with shared logic and two subclass variants:
FreeformRenderer (prose-style) and StructuredRenderer (colon-separated table + Key/Other sections).
A render_report() factory function dispatches based on template variant field.
"""

from __future__ import annotations

import logging
import re
from typing import Callable

from .loader import LoadedTemplate
from .schema import (
    FieldDefinition,
    FieldGroup,
    NOT_DOCUMENTED,
    PLACEHOLDER_PATTERN,
    TemplateSchema,
)

logger = logging.getLogger(__name__)
```

**Constants:**
```python
GUIDANCE_PATTERN = re.compile(
    r"^## Guidance\s*\n.*?(?=^## |\Z)",
    re.MULTILINE | re.DOTALL,
)

SECTION_HEADER_PATTERN = re.compile(r"^## (.+)$", re.MULTILINE)

BLANK_LINE_PATTERN = re.compile(r"\n{4,}")

TABLE_ROW_PATTERN = re.compile(r"\|\s*(.+?)\s*\|\s*\{\{(\w+)\}\}\s*\|")
```

**ReportRenderer base class** (per D-05, D-30, D-31, D-32):
- `render()` method with signature per D-30: `render(self, template: LoadedTemplate, findings: dict, technique: dict, important_fields: list[str] | None = None, rest_normal: bool = False, generate_impression: Callable[[str, str], str] | None = None) -> str`
- Determine effective `interpolate` flag: `template.schema.interpolate_normal or rest_normal` (per D-04)
- Call `_resolve_field_values(template.schema, findings, interpolate)` to get field->value map
- Call `_resolve_groups(template.schema, field_values, findings, interpolate)` to apply group collapse
- Call `_substitute_technique(body, technique)` for `{{technique:*}}` placeholders. For `{{technique:comparison}}`, default to `"None available."` if key absent (per D-29)
- Call `_assemble_body(body, field_values, important_fields)` (subclass override)
- Call `_substitute_measurements(body, technique)` for `{{measurement:*}}` placeholders -- second pass (per two-pass approach). Missing measurements become `NOT_DOCUMENTED`
- Call `_strip_guidance(body)` using GUIDANCE_PATTERN
- Handle impression section: if `template.schema.impression` is True, call `generate_impression(findings_text, clinical_history)` if callable provided, else insert `"(impression not generated)"` per D-21. Log input/output per D-23. Replace the `## COMMENT` section content. If `template.schema.impression` is False, strip entire COMMENT section (header + any content after it until end of string) per D-22
- Call `_convert_headers(body)` to strip `## ` prefix from section headers, keeping text UPPERCASE per D-24
- Call `_cleanup_blank_lines(body)` to collapse 3+ consecutive blank lines to 2 per D-27
- Call `_post_render_validate(body)` to scan for remaining `{{...}}` placeholders and log warnings per D-28
- Return the final string stripped of trailing whitespace

**_resolve_field_values method:**
- For each field in `template.schema.fields`:
  - If `findings.get(field.name)` is not None: use that value (LLM-provided, per D-04)
  - Elif `interpolate` is True: use `field.normal`
  - Elif `field.optional` is True: use `None` (will be omitted -- placeholder line removed)
  - Else: use `NOT_DOCUMENTED`
- Return dict mapping field_name -> str | None

**_resolve_groups method** (per D-01, D-02, D-03):
- Only activates when `interpolate` is True. When False, return field_values unchanged (per D-03)
- For each group in `template.schema.groups`:
  - Find unreported members: those where `findings.get(member) is None`
  - If ALL members unreported: set first member's value to `group.joint_normal`, set remaining members' values to `""` (per D-01)
  - If partially unreported: search `group.partials` for a match where `set(partial.members) == set(unreported)`. If found, set first unreported member's value to `partial.text`, set remaining unreported to `""` (per D-02). If no partial match, leave individual normals in place
- Return updated field_values

**_substitute_technique method:**
- Find all `{{technique:key}}` placeholders in body
- For each, look up `technique.get(key)`. If found, substitute. If not found and key is `"comparison"`, substitute `"None available."` per D-29. Otherwise substitute `NOT_DOCUMENTED`

**_substitute_measurements method:**
- Find all `{{measurement:key}}` placeholders in body
- For each, look up `technique.get(key)` (measurements are passed in the technique dict). If found, substitute. If not found, substitute `NOT_DOCUMENTED`

**_strip_guidance method:**
- Use `GUIDANCE_PATTERN.sub("", body)` to remove guidance section

**_convert_headers method:**
- Use `SECTION_HEADER_PATTERN.sub(r"\1", body)` to strip `## ` prefix

**_cleanup_blank_lines method:**
- Use `BLANK_LINE_PATTERN.sub("\n\n\n", body)` to collapse 4+ newlines to 3 (which visually = 2 blank lines between text)
- Also handle removal of lines that are just whitespace from omitted optional/sex fields: replace lines containing only a placeholder that resolved to None with empty string before blank line collapse

**_post_render_validate method:**
- Use `PLACEHOLDER_PATTERN.findall(body)` to find remaining placeholders
- For each, `logger.warning(f"Unresolved placeholder remaining in output: {{{{{p}}}}}")` per D-28
- Return body unchanged (still return output even with warnings per D-32)

**FreeformRenderer subclass** (per D-06, D-15):
- `_assemble_body(self, body: str, field_values: dict[str, str | None], important_fields: list[str] | None) -> str`
- If `important_fields` is provided and non-empty (per D-15):
  - Identify the FINDINGS section in the body (between `## FINDINGS` and next `## `)
  - Within FINDINGS, collect placeholder lines for important fields and for remaining fields
  - Reorder so important field placeholder lines come first, then remaining in original template order
  - Reconstruct FINDINGS section with reordered lines

  - If `field_values[name]` is None: remove the entire placeholder line (optional/sex-filtered field)
  - If `field_values[name]` is `""`: remove the entire placeholder line (collapsed group member)
  - Otherwise: replace `{{field_name}}` with the value text

**StructuredRenderer subclass** (per D-07, D-10, D-11, D-13, D-16, D-25):
- `_assemble_body(self, body: str, field_values: dict[str, str | None], important_fields: list[str] | None) -> str`
- Parse the markdown table in FINDINGS section using TABLE_ROW_PATTERN
- For each table row, determine status label:
  - If `field_values[field]` is a non-empty string that is NOT `NOT_DOCUMENTED` and NOT the field's normal text: `"See below"`
  - If field was interpolated as normal (value equals the field's normal text): `"Normal"` (per D-11)

  - If `important_fields` provided: route findings text for important fields to Key Findings, others to Other Findings
  - If `important_fields` is None/empty: all findings with values go to Key Findings (per D-17)
  - For each field with findings: output the field label followed by the findings text
  - Fields with `Normal` or `NOT_DOCUMENTED` status: no entry in Key/Other sections (per D-11)
- Replace the original markdown table + Key/Other placeholder content with the rendered plain text

**render_report factory function** (per D-08):
```python
def render_report(
    template: LoadedTemplate,
    findings: dict,
    technique: dict,
    important_fields: list[str] | None = None,
    rest_normal: bool = False,
    generate_impression: Callable[[str, str], str] | None = None,
) -> str:
    variant = getattr(template.schema, "variant", "freeform")
    renderer_map: dict[str, type[ReportRenderer]] = {
        "freeform": FreeformRenderer,
        "structured": StructuredRenderer,
    }
    renderer_cls = renderer_map.get(variant, FreeformRenderer)
    renderer = renderer_cls()
    return renderer.render(
        template, findings, technique,
        important_fields, rest_normal, generate_impression,
    )
```
